# Nature Activity Tracker

A web-based application that encourages users to perform eco-friendly activities and track their environmental impact.

## Features

- User Authentication
- Activity Logging
- Coin Reward System
- Dashboard for Tracking Activities
- Gallery of Nature Activities
- Environmental Awareness Features
- SQLite Database Integration

## Technologies Used

- ASP.NET Core MVC
- C#
- Razor Views (.cshtml)
- Entity Framework Core
- SQLite
- HTML
- CSS
- JavaScript

## Project Structure

- Controllers
- Models
- Views
- Services
- Migrations
- wwwroot

## Activities Supported

- Tree Plantation
- Waste Collection
- Water Conservation
- Energy Saving
- Other Eco-Friendly Activities
